手动更换镜像源

手动更换 Debian 镜像源配置主要分为以下三步：

  1. 用熟悉的文本编辑器打开 /etc/apt/sources.list 文件（需要 sudo 权限），或是直接执行 sudo apt edit-sources 命令。
  2. 替换文件内容为上面任意一个镜像站的源配置代码并保存。
  3. 运行 sudo apt-get update 命令更新软件索引。
